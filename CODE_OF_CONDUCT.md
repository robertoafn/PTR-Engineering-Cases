# Code of Conduct

Este proyecto adopta el [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
como código de conducta.

## Principios resumidos

- Respeto técnico y personal en toda interacción.
- Crítica enfocada en el artefacto, no en la persona.
- Inclusión y apertura a aportes de cualquier nivel de experiencia.
- Cero tolerancia a acoso, discriminación o conducta deshonesta.
