# Glosario

| Término | Definición operativa | Referencia |
|---|---|---|
| FAIR | Findable, Accessible, Interoperable, Reusable | Wilkinson et al., 2016 (DOI:10.1038/sdata.2016.18) |
| SI | Sistema Internacional de Unidades | BIPM, *SI Brochure* 9ª ed. |
| IUPAC | International Union of Pure and Applied Chemistry — nomenclatura química | IUPAC Gold Book |
| ISO 8000 | Estándar de calidad de datos | ISO 8000 series |
| VIM | International Vocabulary of Metrology | JCGM 200:2012 |
| GUM | Guide to the Expression of Uncertainty in Measurement | JCGM 100:2008 |
| Provenance | Cadena documentada de origen, transformaciones y agentes de un artefacto | W3C PROV-O |
| Lineage | Relación `derived_from` entre datasets/artefactos | — |
| Reproducibilidad | Capacidad de obtener el mismo resultado con los mismos datos y métodos | ACM 2020 |
| Sintético | Dato generado por modelo declarado, no observado | — |
| Hipotético | Dato asumido para fines didácticos, sin pretensión de realidad | — |
| Simulado | Dato producido por un simulador de proceso (p. ej. DWSIM) | — |
| Literatura | Dato extraído o derivado de fuente publicada con DOI/ISBN | — |
| Sidecar | Archivo `.meta.yaml` que acompaña a un dataset con sus metadatos | — |
| SemVer | Semantic Versioning 2.0.0 | semver.org |
